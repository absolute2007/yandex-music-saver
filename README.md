# Yandex Music Saver

WinUI 3 desktop app for saving Yandex Music tracks and playlists with `yt-dlp`.

## Requirements

- Windows 10 1809 or newer
- .NET 10 runtime
- `yt-dlp` available in `PATH`
- `ffmpeg` available in `PATH`

## Usage

1. Add one or more Yandex Music links.
2. Select an output folder.
3. Choose browser cookies when access requires authorization.
4. Start the download.

The app supports regular Yandex Music links and shared playlist links.
